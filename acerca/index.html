Acerca

Hola, y gracias por visitar nuestra página :)

Soy madre de dos adorables chiquitos, con un gran interés en la educación y la tecnología. Anteriormente conocido como Familia Windows Phone, Familia Windows es el único sitio dedicado a la categorización y promoción de aplicaciones para Windows 8 y Windows Phone para toda la familia desde una perspectiva de madre.

Mi misión para Familia Windows es :

1 ) Proporcionar a los padres de familia la oportunidad de descubrir aplicaciones para Windows 8 y Windows Phone, que mejoren sus vidas, o eduquen y entretengan a sus hijos.

2 ) Ser miembro activo del ecosistema de Windows y Windows Phone , expresando las necesidades de los padres y los niños a la comunidad de desarrolladores con el fin de fomentar el desarrollo de aplicaciones para la familia de alta calidad.

3 ) Apoyar a la comunidad de desarrolladores , ofreciendo oportunidades de promocionar dentro de nuestra plataforma, para que puedan acercarse a miles de padres de familia con dispositivos Windows en mercados de habla inglés y español.

Por favor, no dude en contactarme si tiene alguna pregunta, sugerencia o comentario.

¡Saludos!
